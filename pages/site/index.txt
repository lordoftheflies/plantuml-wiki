<nodisp>TITLE: Open-source tool that uses simple textual descriptions to draw UML diagrams.</nodisp>
<nodisp>DESC: Easily create UML Diagrams from simple textual description. There are also numerous kind of available diagrams. It's also possible to export images in PNG, LaTeX, EPS, SVG.</nodisp>

====== PlantUML in a nutshell ======

**PlantUML** is a component that allows to quickly write :

  * [[sequence-diagram|Sequence diagram]]
  * [[use-case-diagram|Usecase diagram]]
  * [[class-diagram|Class diagram]]
  * [[activity-diagram-beta|Activity diagram]] (here is [[activity-diagram-legacy|the legacy syntax)]]
  * [[component-diagram|Component diagram]]
  * [[state-diagram|State diagram]]
  * [[object-diagram|Object diagram]]
  * [[deployment-diagram|Deployment diagram]] {{:site:beta-icon.gif?nolink|}}
  * [[timing-diagram|Timing diagram]] {{:site:beta-icon.gif?nolink|}}

The following non-UML diagrams are also supported:

  * [[salt|Wireframe graphical interface]]
  * [[archimate-diagram|Archimate diagram]]
  * [[activity-diagram-beta#sdl|Specification and Description Language (SDL)]]
  * [[ditaa|Ditaa diagram]]
  * [[gantt-diagram|Gantt diagram]] {{:site:beta-icon.gif?nolink|}}
  * [[ascii-math|Mathematic with AsciiMath or JLaTeXMath notation]]

Diagrams are defined using a simple and intuitive language. ([[http://plantuml.com/PlantUML_Language_Reference_Guide.pdf|see PlantUML Language Reference Guide]]).

New users can read the [[starting|quick start page]]. There is also a [[faq|F.A.Q. page]]. PlantUML can be used within [[running|many other tools]].

Images can be generated in PNG, [[svg|in SVG]] or [[latex|in LaTeX]] format. It is also possible to generate [[ascii-art|ASCII art diagrams]] (only for sequence diagrams).

**# online generations:**
  * Since ever //4242424242//
  * Last 24 hours //42424242//
  * Last hour //4242//
  * Last minute //42//

^ __Example__: \\  //(You can edit the text if you want, the diagram will be dynamically updated!)// ^ __describes the following diagram :__^
| ''Bob->Alice : hello''    | {{:site:syffkj2rkt3coknelr1io4zdosa70000.png?nolink|}}     |

This example is working thanks to the [[http://www.plantuml.com/plantuml|online demo server]].
